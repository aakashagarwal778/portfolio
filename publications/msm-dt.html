Regime-Conditional Portfolio Allocation: A Markov-Switching Decision Tree Framework

17 years out-of-sample · 35 walk-forward folds SPY · EEM · TLT · GLD 25 bps transaction costs January 2025
Download PDF ↗ View code on GitHub ↗
Abstract

A systematic, regime-conditional framework for multi-asset portfolio allocation. A three-state Gaussian Hidden Markov Model identifies latent market regimes from a compact feature vector comprising realised volatility, cross-asset correlation, and credit spread. Regime-conditional weights are constructed analytically — Sharpe-maximising mean-variance under Ledoit-Wolf covariance shrinkage for Risk-On, risk parity for Risk-Off, and a GLD/TLT flight-to-quality allocation for Crisis. A Decision Tree learns the continuous mapping from feature space to weights, eliminating the hard switching artefact of discrete regime assignment. Evaluated on 20 years of daily data across four asset classes using a strict walk-forward protocol with 35 out-of-sample folds and 25 basis point transaction costs.
